Sous linux :
Pour compiler il suffit de se placer dans le dossier Agenda, ensuite taper la commande
ant

Pour supprimer tous les fichiers compilent, il faut taper la commande 
ant clean

Pour exécuter l'application console(AppConsole.java) dans entretient, il faut taper la commande 
source appEnt.sh

Pour exécuter l'application console(AppConsole.java) dans service informatique si, il faut taper la commande 
source appSI.sh

Pour exécuter l'interface graphique(GUInteface.java), il faut taper la commande 
source appG.sh
